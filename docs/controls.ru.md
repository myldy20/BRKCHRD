# Управление BRKCHRD 0.6.0 — Русский

## Раскладка TrimUI Brick

В протестированном профиле Knulli:

- передние светящиеся слева/справа — октава вниз/вверх;
- задняя L1 — переключение экрана крестовины;
- задняя L2 — удержание альтернативного слоя;
- задние R1/R2 — временные банки аккордов;
- ABXY — функциональная раскладка аккордов.

Если прошивка определяет задние кнопки в обратном физическом порядке, используйте `СМЕНА L1/L2` или `СМЕНА R1/R2`.

## Глобальное управление

| Кнопка | Действие |
| --- | --- |
| Передние светящиеся слева/справа | Октава вниз/вверх |
| L1, нажатие | АККОРД → ЗВУК → ПЕРФ FX |
| L2, удержание | Альтернативная палитра, слой параметров или банк ПЕРФ FX |
| R1/R2, удержание | Временный назначенный банк аккордов |
| ABXY, удержание | Играть аккорды |
| Select | Открыть или закрыть настройки |
| Start, коротко | Следующий ПЭД / БОЙ / АРП / ПУЛЬС |
| Start, долго | Снять все ноты |
| Start + Select | Сохранить и выйти |

При `ПЕРФ FX = ВЫКЛ` L1 переключает только АККОРД ↔ ЗВУК.

## LANGUAGE / ЯЗЫК

- `EN` — английский интерфейс;
- `RU` — русский интерфейс с настоящими кириллическими символами встроенного шрифта.

Язык меняется сразу и сохраняется после перезапуска. Названия пресетов и символы аккордов остаются общепринятыми.

## Режим АККОРД

Обычная и L2-палитра независимо назначаются как КЛАССИКА, РАСШИР. или ТЕМНАЯ. Удерживайте L2 во время работы крестовиной, чтобы использовать вторую палитру.

### DPAD АККОРД = НАЖАТИЕ

- нажмите направление, чтобы сохранить его окраску;
- после отпускания и перехода на другие экраны окраска остаётся активной;
- повторное нажатие того же направления в той же палитре возвращает ОСНОВУ;
- окраска из L2-палитры остаётся после отпускания L2.

### DPAD АККОРД = УДЕРЖ.

- отпущенная крестовина означает ОСНОВУ;
- окраска действует только пока направление удерживается;
- после отпускания аккорд сразу возвращается к ОСНОВЕ;
- нажатие или отпускание второй оси переводит жест в диагональ или обратно;
- уже звучащий аккорд перестраивается при каждом изменении жеста;
- L2 выбирает альтернативную палитру только на время удержания.

УДЕРЖ. не стирает окраску, сохранённую режимом НАЖАТИЕ. При возврате к НАЖАТИЕ она снова станет активной. Активная окраска уже показана под левой таблицей, поэтому отдельное всплывающее окно при жестах крестовины не появляется.

## Режим ЗВУК

Обычный слой:

| Выбор вверх/вниз | Изменение влево/вправо |
| --- | --- |
| Пресет | Предыдущий / следующий звук |
| Тон | Темнее / ярче |
| Тело | Меньше / больше плотности |
| Движение | Меньше / больше движения |

Слой при удержании L2:

| Выбор вверх/вниз | Изменение влево/вправо |
| --- | --- |
| Атака | Быстрее / медленнее |
| Затухание | Короче / длиннее |
| Ширина | Уже / шире |
| BPM | Ниже / выше темп |
| Режим игры | Предыдущий / следующий |

Удержание влево/вправо ускоряет редактирование. ABXY и R1/R2 продолжают работать.

## Режим ПЕРФ FX

Удерживайте направление, чтобы временно подменить FX1/FX2 комбинацией из двух эффектов. После отпускания исходная цепочка восстанавливается. L2 открывает второй банк жестов.

## Настройки

- вверх/вниз — выбрать строку;
- влево/вправо — изменить значение;
- Select — закрыть и сохранить;
- Start+Select — сохранить и выйти.

Основные переключатели: ВЕДЕНИЕ, ПЕРФ FX, АНИМАЦИЯ, перестановка задних кнопок, `LANGUAGE / ЯЗЫК`, `DPAD АККОРД` и `ПАЛИТРА UI`.

`ПАЛИТРА UI` предлагает варианты АМЕТИСТ, ЛОГО, ОКЕАН, УГЛИ и МОНО. Оформление меняется сразу, сохраняется вместе с настройками и при следующем запуске перекрашивает заставку. ЛОГО использует исходные цвета проекта: угольный `#111514` и кремовый `#f1e3c5`.
