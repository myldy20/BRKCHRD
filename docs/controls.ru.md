# Управление — Русский

## Реальная раскладка TrimUI Brick

BRKCHRD 0.3 использует не абстрактную раскладку Xbox-контроллера, а фактические сигналы, которые Knulli отдаёт на TrimUI Brick:

- две светящиеся клавиши на лицевой панели приходят как левый и правый shoulder;
- задние L1 и L2 образуют один левый канал;
- задние R1 и R2 образуют один правый канал;
- подписи ABXY в SDL переставлены относительно надписей на корпусе Brick.

Приложение учитывает это напрямую.

## Основная схема

| Физическая клавиша | Действие |
| --- | --- |
| передняя светящаяся слева | октава вниз |
| передняя светящаяся справа | октава вверх |
| любая задняя L: L1 или L2 | следующий режим левой панели |
| любая задняя R: R1 или R2 | следующий режим правой панели |
| Start | следующий режим PAD / STRUM / ARP / PULSE |
| короткий Select | Latch вкл./выкл. |
| удерживать Select | снять Latch и немедленно остановить все ноты |
| Start + Select | сохранить настройки и выйти |

Последняя нажатая задняя сторона становится активной: её панель получает яркую двойную рамку, а D-pad управляет именно ей.

## Левая панель

Задние L1/L2 циклически переключают:

1. **CLASSIC** — базовые окраски аккордов;
2. **EXTENDED** — расширенные джазовые и модальные окраски;
3. **DARK** — power, open, cluster, tritone и другие тяжёлые варианты;
4. **SOUND** — движок и макропараметры синтеза;
5. **SYSTEM** — тональность, отображение октавы и интенсивность анимации.

В режимах CLASSIC / EXTENDED / DARK D-pad выбирает клетку 3×3. Повторное нажатие выбранного направления возвращает BASE. Если аккорд уже зажат, D-pad временно меняет его окраску; после отпускания возвращается заранее выбранная.

В SOUND и SYSTEM:

- D-pad Up/Down выбирает строку;
- D-pad Left/Right меняет значение;
- удержание Left/Right ускоряет изменение.

## Правая панель

Задние R1/R2 циклически переключают:

1. **CORE** — основная банка;
2. **DIATONIC+** — дополнительные диатонические ступени;
3. **BORROWED** — заимствованные минорные и модальные ступени;
4. **FX 1** — первый эффект;
5. **FX 2** — второй эффект;
6. **MASTER** — общий уровень и схема сигнала.

При переходе в CORE / DIATONIC+ / BORROWED соответствующая банка становится активной и остаётся активной во время редактирования эффектов.

В FX 1 и FX 2:

- Up/Down выбирает TYPE, AMOUNT или COLOUR;
- Left/Right меняет параметр.

В MASTER Left/Right регулирует общий уровень.

## Физические ABXY и аккорды

Интерфейс теперь показывает буквы с корпуса Brick, а не логические SDL-названия.

| Физическая позиция | Клавиша | CORE | DIATONIC+ | BORROWED |
| --- | --- | --- | --- | --- |
| снизу | B | I | ii | i |
| справа | A | V | iii | iv |
| слева | Y | vi | vii° | ♭VI |
| сверху | X | IV | ♭VII | ♭III |

ABXY играют в любом режиме обеих панелей: можно менять звук или эффект и сразу слушать результат.

## Окраски

### CLASSIC

Up — major/minor flip; Up-right — dominant/minor 7; Right — maj7/min7; Down-right — add9/min9; Down — sus4; Down-left — 6/sus2; Left — dark/dim; Up-left — augmented.

### EXTENDED

ADD11, DOM9, 6/9, MIN11, 7SUS, HALF-DIM, mMAJ7 и LYDIAN.

### DARK

POWER, CRUNCH 7#9, QUARTAL, CLUSTER add♭9, OPEN без терции, MIN6, TRITONE и AUGMAJ7.

## Desktop-раскладка

| Клавиатура | TrimUI Brick |
| --- | --- |
| Z / X / A / S | нижняя B / правая A / левая Y / верхняя X |
| стрелки | D-pad |
| Q | следующий режим левой панели |
| W | следующий режим правой панели |
| E / R | октава вниз / вверх |
| Enter | Start |
| Space | Select |
| Esc | выход |

## Диагностика кнопок

При запуске приложение записывает в `brkchrd.log` имя контроллера, SDL mapping, controller button events и первые raw button/axis events. Это позволяет точно диагностировать отличия других версий Knulli без гадания по названиям кнопок.
