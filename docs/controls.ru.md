# Управление — Русский

## Логика инструмента

BRKCHRD разделяет управление на два слоя:

1. **Страница PLAY:** D-pad заранее выбирает или временно меняет окраску аккорда. Тональность и октава здесь никогда не переключаются.
2. **Страницы редактирования:** D-pad выбирает и меняет параметры звука, эффектов и системы.

A/B/X/Y играют на любой странице — можно редактировать звук и тут же проверять результат.

## Кнопки аккордов и банки

| Кнопка | Основная банка | Удерживать заднюю L | Удерживать заднюю R |
| --- | --- | --- | --- |
| A | I | ii | i |
| B | V | iii | iv |
| X | vi | vii° | ♭VI |
| Y | IV | ♭VII | ♭III |

У задних L/R две функции:

- **удерживать вместе с аккордом:** выбрать дополнительную банку;
- **коротко нажать без аккорда:** перейти на предыдущую/следующую страницу.

Если после L/R нажата A/B/X/Y, переход страницы отменяется — можно спокойно готовить альтернативный аккорд.

## Окраски аккордов

На странице PLAY нажми направление D-pad — оно останется выбранным для следующего аккорда. Повторное нажатие того же направления возвращает BASE.

Если аккорд уже звучит, D-pad меняет его сразу. После отпускания D-pad инструмент возвращается к заранее выбранной окраске.

### CLASSIC

Up — смена major/minor; Up-right — доминантсептаккорд/min7; Right — maj7/min7; Down-right — add9/min9; Down — sus4; Down-left — 6/sus2; Left — dark/dim; Up-left — augmented.

### EXTENDED — светящаяся FX1

ADD11, DOM9, 6/9, MIN11, 7SUS, HALF-DIM, mMAJ7 и LYDIAN.

### DARK — светящаяся FX2

POWER, CRUNCH 7#9, QUARTAL, CLUSTER add♭9, OPEN без терции, MIN6, TRITONE и AUGMAJ7.

Одновременное нажатие FX1+FX2 возвращает CLASSIC + BASE.

## Глобальное управление

| Кнопка | Действие |
| --- | --- |
| Start | следующий режим PAD / STRUM / ARP / PULSE |
| Select | включить/выключить Latch |
| Start + Select | сохранить и выйти |
| короткое L/R | предыдущая/следующая страница |
| D-pad Up/Down | выбрать строку вне PLAY |
| D-pad Left/Right | изменить значение вне PLAY |

При удержании Left/Right изменение ускоряется примерно через 0,85 и 2,2 секунды.
