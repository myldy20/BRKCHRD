# Управление BRKCHRD 0.4.0 — Русский

## Подтверждённая раскладка TrimUI Brick

В использованной сборке Knulli кнопки определяются раздельно:

- передняя светящаяся слева — SDL left shoulder;
- передняя светящаяся справа — SDL right shoulder;
- задняя L1 — SDL left-stick button;
- задняя L2 — SDL left trigger;
- задняя R1 — SDL right-stick button;
- задняя R2 — SDL right trigger;
- D-pad — направления hat;
- буквы ABXY на экране соответствуют физическим надписям на Brick, а не логическим именам SDL.

Если конкретная прошивка выдаёт L1/L2 или R1/R2 в обратном физическом порядке, используй настройки `SWAP L1/L2` и `SWAP R1/R2`.

## Основная схема

| Физическая клавиша | Действие |
| --- | --- |
| передняя светящаяся слева | октава вниз |
| передняя светящаяся справа | октава вверх |
| удерживать L1 | временный альтернативный слой текущего режима D-pad |
| нажать L2 | цикл `CHORD → SOUND → PERF FX` |
| удерживать R1 | настроенная дополнительная банка аккордов 1 |
| удерживать R2 | настроенная дополнительная банка аккордов 2 |
| ABXY | играть функциональные аккорды |
| коротко Start | цикл PAD / STRUM / ARP / PULSE |
| удерживать Start | остановить все ноты |
| Select | открыть или закрыть Settings |
| Start + Select | сохранить и выйти |

Latch намеренно удалён. После отпускания последней удерживаемой ABXY аккорд отпускается.

## Режим CHORD

D-pad выбирает окраску аккорда. Выбор хранится отдельно от палитры, которая сейчас показана на экране.

- нажми направление — окраска сохранится;
- повторно нажми то же направление при той же отображаемой палитре — вернётся BASE;
- сыграй любой аккорд ABXY — к нему применяется сохранённая окраска;
- перейди в SOUND или PERF FX — сохранённая окраска продолжает действовать;
- удерживай L1 — откроется альтернативная палитра;
- отпусти L1 — вернётся основная палитра, но выбранная окраска не сбросится.

В Settings независимо задаются:

- `BASE COLOUR` — палитра без L1;
- `L1 COLOUR` — палитра при удержании L1.

Для каждого слоя можно выбрать CLASSIC, EXTENDED или DARK.

### Направления CLASSIC

| Направление | Окраска |
| --- | --- |
| Up-left | augmented |
| Up | смена major/minor |
| Up-right | dominant 7 / minor 7 |
| Left | dark minor / diminished |
| Right | major 7 / minor 7 |
| Down-left | sixth / sus2 |
| Down | sus4 |
| Down-right | add9 / minor 9 |

### EXTENDED

ADD11, DOM9, 6/9, MIN11, 7SUS, HALF-DIM, mMAJ7 и LYDIAN.

### DARK

POWER, CRUNCH 7#9, QUARTAL, CLUSTER add♭9, OPEN без терции, MIN6, TRITONE и AUGMAJ7.

## Режим SOUND

Один раз нажми L2 из CHORD. ABXY продолжают играть, а сохранённая окраска аккорда не меняется.

### Обычный слой

- Preset
- Tone
- Body
- Motion

### Слой при удержании L1

- Attack
- Release
- Spread
- BPM
- Play Mode

D-pad Up/Down выбирает строку. Left/Right меняет значение. Долгое удержание Left/Right ускоряет изменение.

L1 работает моментально: после отпускания возвращается основной набор параметров, все изменения сохраняются.

## Режим PERF FX

Ещё раз нажми L2 из SOUND. Теперь D-pad становится контроллером моментальных сценических эффектов.

- удерживай направление D-pad — включается назначенная комбинация двух эффектов;
- переведи D-pad в другое направление — комбинация меняется сразу;
- отпусти D-pad — исходные базовые настройки FX1 и FX2 полностью восстанавливаются;
- удерживай L1 — используется вторая, более экстремальная банка эффектов;
- отпусти L1 при удерживаемом D-pad — инструмент вернётся к обычной банке PERF FX.

### Обычная банка

Reverse-ish, Stutter, Chop, Crush, Drive, Wash, Deep Echo и Phase.

### Банка при удержании L1

Abyss, Ratetrap, Glitch, Broken, Doom, Shimmer-ish, Tunnel и Alien.

Названия обозначают сценические жесты. В текущем движке они реализованы комбинациями Chorus, Phaser, Tremolo, Drive, Crusher, Delay и Reverb. PERF FX действует только во время удержания и не перезаписывает базовые эффекты.

## Банки аккордов

В Settings независимо задаются:

- `BASE BANK` — когда R1/R2 не зажаты;
- `R1 BANK` — при удержании R1;
- `R2 BANK` — при удержании R2.

Для каждого слоя можно выбрать CORE, DIATONIC+ или BORROWED.

| Физическая позиция | Клавиша | CORE | DIATONIC+ | BORROWED |
| --- | --- | --- | --- | --- |
| снизу | B | I | ii | i |
| справа | A | V | iii | iv |
| слева | Y | vi | vii° | ♭VI |
| сверху | X | IV | ♭VII | ♭III |

R1 и R2 действуют только во время удержания. После отпускания возвращается BASE BANK. Если аккорд уже звучит, он немедленно перестраивается в выбранную банку.

## Settings

Нажми Select, чтобы открыть полноэкранное меню. D-pad Up/Down выбирает строку, Left/Right меняет значение.

Доступные настройки:

1. Основная палитра окраски
2. Палитра при удержании L1
3. Основная банка аккордов
4. Банка при удержании R1
5. Банка при удержании R2
6. Тип FX1
7. Глубина FX1
8. Характер FX1
9. Тип FX2
10. Глубина FX2
11. Характер FX2
12. Тональность
13. Октава
14. Интенсивность анимации
15. Поменять местами L1/L2
16. Поменять местами R1/R2

Select закрывает Settings и сохраняет настройки. Start+Select сохраняет и выходит из программы.

## Передние клавиши октавы

Светящиеся передние клавиши меняют октаву вниз и вверх. Если аккорд зажат, он сразу перестраивается в новой октаве.

## Desktop-раскладка

| Клавиатура | Действие на консоли |
| --- | --- |
| Z / X / A / S | физические B / A / Y / X |
| стрелки | D-pad |
| удерживать Q | L1 |
| нажать W | L2 |
| удерживать E | R1 |
| удерживать R | R2 |
| [ / ] | октава вниз / вверх |
| Enter | Start |
| Space | Select |
| Esc | выход |

## Диагностика

`brkchrd.log` содержит SDL mapping, raw-события кнопок и осей и ожидаемые роли v0.4. При проблеме с порядком задних клавиш сначала используй две настройки SWAP, а не переназначай весь контроллер Knulli.
