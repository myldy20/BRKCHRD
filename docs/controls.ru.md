# Управление BRKCHRD 0.5.1 — Русский

## Подтверждённая раскладка TrimUI Brick

В протестированном профиле Knulli:

- передняя светящаяся слева — SDL left shoulder;
- передняя светящаяся справа — SDL right shoulder;
- задняя L1 — SDL left-stick button;
- задняя L2 — SDL left trigger;
- задняя R1 — SDL right-stick button;
- задняя R2 — SDL right trigger;
- буквы ABXY соответствуют надписям на корпусе Brick, а не логическим буквам SDL.

Если конкретная прошивка определяет задние кнопки в другом физическом порядке, используйте `SWAP L1/L2` или `SWAP R1/R2`.

## Глобальное управление

| Кнопка | Действие | Результат |
| --- | --- | --- |
| Передняя светящаяся слева | Нажать | Октава вниз |
| Передняя светящаяся справа | Нажать | Октава вверх |
| L1 | Нажать | CHORD → SOUND → PERF FX |
| L2 | Удерживать | Альтернативный слой текущего экрана |
| R1 | Удерживать | Временный банк, назначенный R1 |
| R2 | Удерживать | Временный банк, назначенный R2 |
| ABXY | Удерживать | Играть функциональные аккорды |
| Select | Нажать | Открыть или закрыть Settings |
| Start | Коротко | Следующий PAD / STRUM / ARP / PULSE |
| Start | Долго | Снять все ноты |
| Start + Select | Вместе | Сохранить и выйти |

При `PERF FX = OFF` L1 переключает только CHORD ↔ SOUND.

## Режим CHORD

Направления крестовины выбирают одну сохранённую гармоническую окраску. Она продолжает действовать в SOUND, PERF FX и Settings.

- нажать новое направление — выбрать окраску;
- повторно нажать уже выбранное направление — вернуться к BASE;
- удерживать L2 во время выбора — использовать альтернативную палитру;
- обычная и L2-палитра независимо назначаются как CLASSIC, EXTENDED или DARK.

## Режим SOUND

Обычный слой:

| Выбор вверх/вниз | Изменение влево/вправо |
| --- | --- |
| Preset | Предыдущий / следующий звук |
| Tone | Темнее / ярче |
| Body | Меньше / больше плотности |
| Motion | Меньше / больше движения |

Слой при удержании L2:

| Выбор вверх/вниз | Изменение влево/вправо |
| --- | --- |
| Attack | Быстрее / медленнее атака |
| Release | Короче / длиннее хвост |
| Spread | Уже / шире |
| BPM | Ниже / выше темп |
| Play Mode | Предыдущий / следующий режим |

Если удерживать влево или вправо, изменение ускоряется. ABXY остаются активными во время редактирования.

## Режим PERF FX

Удерживайте направление, чтобы временно включить комбинацию из двух сценических эффектов. Отпустите крестовину — исходные FX1/FX2 восстановятся. Удержание L2 открывает второй, более экстремальный банк.

Центральное положение означает отсутствие временной подмены, но не отключает обычные базовые эффекты.

## Модификаторы банков

R1 и R2 работают только при удержании. Для каждой кнопки назначается CORE, DIATONIC+ или BORROWED. Если удерживаются обе, приоритет у R2. Экран и звучащий аккорд обновляются сразу.

## Навигация в Settings

- вверх/вниз — выбрать строку;
- влево/вправо — изменить значение;
- удержание влево/вправо — ускоренный повтор там, где он нужен;
- Select — закрыть и сохранить;
- Start+Select — сохранить и выйти.

В 0.5.1 все значения выровнены справа и не касаются рамки.

## Клавиатура для теста на компьютере

| Клавиша | Эквивалент на консоли |
| --- | --- |
| Z / X / A / S | Физические B / A / Y / X |
| `[` / `]` | Октава вниз / вверх |
| Q | L1, переключение режима |
| W | L2, альтернативный слой |
| E / R | R1 / R2, банки |
| Стрелки | Крестовина |
| Enter | Start |
| Space | Select |
| Escape | Выход |
