# Руководство BRKCHRD 0.3.0 — Русский

## Что это за инструмент

BRKCHRD — не справочник аккордов и не трекер. Это инструмент для живого исполнения гармонии на TrimUI Brick. Четыре лицевые ABXY выбирают функциональные аккорды, D-pad задаёт их окраску, а две независимые половины интерфейса позволяют менять гармоническую логику, синтез и эффекты, не прекращая игру.

## Первая минута

1. Запусти BRKCHRD. Слева откроется CLASSIC, справа CORE.
2. Нажми физические B, A, Y и X. В CORE они играют I, V, vi и IV.
3. Нажми любую заднюю L два раза: левая панель перейдёт CLASSIC → EXTENDED → DARK.
4. Нажми D-pad Up. В DARK заранее выберется POWER.
5. Сыграй любой аккорд: он сразу начнётся как power chord.
6. Ещё раз нажми Up, чтобы вернуть BASE.
7. Передняя светящаяся левая клавиша опускает октаву, правая поднимает.
8. Любая задняя R переключает правую панель между банками аккордов, FX 1, FX 2 и MASTER.
9. Start меняет PAD / STRUM / ARP / PULSE. Короткий Select включает Latch.

## Две независимые панели

Интерфейс всегда состоит из двух равных панелей. Последняя нажатая задняя сторона получает яркую двойную рамку и управление D-pad.

### Левая панель

Задние L1 и L2 дают один и тот же левый сигнал и циклически переключают:

- CLASSIC;
- EXTENDED;
- DARK;
- SOUND;
- SYSTEM.

CLASSIC, EXTENDED и DARK определяют набор из восьми направленных окрасок плюс BASE. SOUND открывает движок и макропараметры. SYSTEM содержит тональность, отображение октавы и уровень анимации интерфейса.

### Правая панель

Задние R1 и R2 дают один и тот же правый сигнал и циклически переключают:

- CORE;
- DIATONIC+;
- BORROWED;
- FX 1;
- FX 2;
- MASTER.

Переход в одну из первых трёх секций выбирает активную банку. Она сохраняется при переходе к эффектам.

## Аккордовые кнопки

На Brick буквы расположены так:

```text
        X
    Y       A
        B
```

В основной банке:

```text
        X = IV
    Y = vi    A = V
        B = I
```

Интерфейс специально показывает физические буквы Brick, а не логические SDL-названия, которые Knulli переставляет попарно.

## Окраска аккорда

В любом цветовом режиме или при открытой банке аккордов D-pad может заранее выбрать окраску. Повторное нажатие того же направления возвращает BASE.

Если удерживать аккорд и нажать D-pad, окраска меняется немедленно и временно. После отпускания направления возвращается заранее выбранная окраска.

### CLASSIC

Базовые minor/major flip, септаккорды, add9, sus, sixth, diminished и augmented.

### EXTENDED

ADD11, DOM9, 6/9, MIN11, 7SUS, HALF-DIM, mMAJ7 и LYDIAN.

### DARK

POWER, CRUNCH 7#9, QUARTAL, CLUSTER, OPEN, MIN6, TRITONE и AUGMAJ7.

## SOUND

Строки:

- ENGINE;
- TONE;
- BODY;
- MOTION;
- ATTACK;
- RELEASE;
- SPREAD;
- BPM;
- PLAY MODE.

Up/Down выбирают строку, Left/Right меняют значение. Удержание Left/Right ускоряет изменение. ABXY продолжают играть, поэтому результат слышен сразу.

## FX 1 и FX 2

Каждый последовательный слот имеет:

- TYPE — алгоритм;
- AMOUNT — глубина;
- COLOUR — второй характерный параметр.

Доступны Off, Chorus, Phaser, Tremolo, Drive, Crusher, Delay и Reverb.

## MASTER

MASTER регулирует финальный уровень после двух эффектов и показывает цепочку:

```text
SYNTH → FX 1 → FX 2 → MASTER
```

## SYSTEM

- KEY — тональность;
- OCTAVE — текущее смещение; меняется передними светящимися кнопками;
- UI MOTION — OFF / LOW / FULL;
- VERSION — версия приложения.

Отдельных пунктов RESET COLOUR и PANIC больше нет. BASE возвращается повторным нажатием направления D-pad. Для аварийной остановки удерживай Select.

## Latch и остановка нот

Короткий Select включает или выключает Latch. При Latch последний аккорд остаётся звучать после отпускания ABXY.

Удержание Select примерно 0,85 секунды:

- выключает Latch;
- останавливает все голоса;
- очищает состояние зажатых аккордов;
- показывает `ALL NOTES OFF`.

## Сохранение и выход

Start + Select сохраняет:

- тональность и октаву;
- текущие режимы панелей;
- банк и палитру;
- выбранную окраску;
- движок и макропараметры;
- оба эффекта;
- BPM, режим исполнения и Latch;
- интенсивность анимации.

После сохранения приложение закрывается.

## Диагностика

`brkchrd.log` содержит имя контроллера, SDL mapping и ограниченный список raw button/axis events. Если новая версия Knulli снова изменит раскладку, этот лог позволит увидеть фактические коды без отдельной диагностической программы.
