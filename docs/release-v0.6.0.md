# BRKCHRD 0.6.0

BRKCHRD 0.6.0 is the first fully stabilised public release for TrimUI Brick.

## Download

Choose **one** package:

- `brkchrd-v0.6.0-portmaster.zip` — Knulli / PortMaster;
- `brkchrd-v0.6.0-nextui-tg5040.zip` — NextUI on `tg5040` devices.

Do not use GitHub's green **Code → Download ZIP** button: that archive contains source code, not the ready-to-install application.

## Highlights

- five persistent UI palettes: AMETHYST, LOGO, OCEAN, EMBER and MONO;
- palette-aware startup logo;
- chord-colour D-pad gestures no longer cover useful labels with a popup;
- English and Russian UI and documentation;
- sixteen synthesis presets and PAD / STRUM / ARP / PULSE modes;
- hardened audio path, bounded output, DC blocking and transition ramps;
- exhaustive 2,219-scenario DSP audit;
- native AArch64 PortMaster and NextUI packages.
