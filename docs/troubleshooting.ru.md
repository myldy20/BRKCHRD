# Диагностика — Русский

## BRKCHRD не появился в Ports

Проверь пути и права:

```bash
ssh trimui 'ls -l "/userdata/roms/ports/BRKCHRD.sh" /userdata/roms/ports/brkchrd/*.aarch64'
```

Исправить права:

```bash
ssh trimui 'chmod +x "/userdata/roms/ports/BRKCHRD.sh" /userdata/roms/ports/brkchrd/*.aarch64 && sync'
```

После этого обнови список игр или перезагрузи консоль.

## Программа сразу закрывается

```bash
ssh trimui 'cat /userdata/roms/ports/brkchrd/conf/brkchrd.log'
```

Типовые причины: отсутствует бинарник, неправильная архитектура, ошибка инициализации SDL audio/video или неполное окружение PortMaster.

## Передние и задние кнопки делают не то

В 0.3 ожидается фактическая раскладка TrimUI Brick на Knulli:

- передняя светящаяся левая — octave down;
- передняя светящаяся правая — octave up;
- задние L1 и L2 — один левый канал, переключающий режим левой панели;
- задние R1 и R2 — один правый канал, переключающий режим правой панели.

Посмотри первые строки лога:

```bash
ssh trimui 'sed -n "1,120p" /userdata/roms/ports/brkchrd/conf/brkchrd.log'
```

Ищи:

```text
controller:
mapping:
controller button down:
raw button down:
raw axis:
```

Приложение записывает ограниченное число сырых событий, чтобы лог не рос бесконечно.

## Перепутаны X/Y или A/B

Интерфейс 0.3 показывает физические надписи Brick:

```text
        X
    Y       A
        B
```

При этом внутри SDL Knulli может называть нижнюю B как A, правую A как B, левую Y как X и верхнюю X как Y. Это нормально: BRKCHRD переводит позиционную SDL-разметку обратно в физические подписи.

Если активируется не тот квадрат по позиции, нужен `brkchrd.log`; это уже другая mapping-конфигурация прошивки.

## D-pad меняет не тот параметр

D-pad принадлежит последней выбранной стороне. Яркая двойная рамка показывает фокус:

- после задней L активна левая панель;
- после задней R активна правая панель.

В CLASSIC / EXTENDED / DARK и в банках аккордов D-pad выбирает окраску. В SOUND / SYSTEM / FX он выбирает и редактирует параметры.

## Как вернуть BASE

В цветовом режиме повторно нажми уже выбранное направление D-pad. Отдельного пункта RESET COLOUR больше нет.

## Как аварийно остановить звук

Удерживай Select примерно 0,85 секунды. BRKCHRD выключит Latch, остановит все голоса и покажет `ALL NOTES OFF`.

## Треск или высокая нагрузка

- снизь Amount эффектов, особенно двух Delay/Reverb подряд;
- не сочетай максимальный Release с плотными шестинотными аккордами при проверке;
- проверь нагрузку сначала с FX Off;
- сравни динамик и наушники;
- приложи к баг-репорту движок, окраску, режим, FX и фактическую нагрузку.

Анимация интерфейса не влияет на аудиограф, но для проверки можно поставить SYSTEM → UI MOTION → OFF.

## Сброс настроек

```bash
ssh trimui 'mv /userdata/roms/ports/brkchrd/conf/brkchrd.cfg /userdata/roms/ports/brkchrd/conf/brkchrd.cfg.bak'
```

Следующий запуск создаст заводское состояние.

## Нормальный баг-репорт

Нужны:

- версия Knulli;
- имя установочного архива;
- `brkchrd.log`;
- реакция каждой физической кнопки;
- фото экрана с наложением или обрезанием текста;
- динамик или наушники;
- приблизительная задержка;
- треск и нагрузка;
- активные левый и правый режимы.
