# Установка BRKCHRD 0.5.1 в NextUI

BRKCHRD выпускается как нативный Pak для платформы `tg5040`. Релизный архив называется:

```text
brkchrd-v0.5.1-nextui-tg5040.zip
```

## Структура пакета

Внутри архива:

```text
Tools/tg5040/BRKCHRD.pak/
Tools/tg5040/.media/BRKCHRD.png
```

В `.pak` находятся launcher, AArch64-бинарник, документация и лицензии. Соседний файл `.media/BRKCHRD.png` отображается в карточке инструмента NextUI.

## Установка через SD-карту

1. Выключите устройство и подключите карту к компьютеру.
2. Распакуйте архив в корень карты, сохранив путь `Tools/tg5040/`.
3. Проверьте:

```text
SD/Tools/tg5040/BRKCHRD.pak/launch.sh
SD/Tools/tg5040/BRKCHRD.pak/brkchrd-sdl.aarch64
SD/Tools/tg5040/.media/BRKCHRD.png
```

4. Верните карту и откройте раздел Tools. BRKCHRD должен появиться с фирменным изображением.

## Постоянные данные

Launcher использует каталог пользовательских данных NextUI и не пишет настройки внутрь Pak:

```text
SD/.userdata/tg5040/brkchrd/brkchrd.cfg
SD/.userdata/tg5040/logs/BRKCHRD.txt
```

Поэтому при обновлении Pak пользовательские настройки сохраняются.

## Обновление

Распакуйте новый архив поверх существующего дерева `Tools/tg5040/`. Не удаляйте `.userdata`, если не нужен полный сброс.

Для сброса только BRKCHRD переименуйте:

```text
SD/.userdata/tg5040/brkchrd/brkchrd.cfg
```

## Удаление

Удалите:

```text
SD/Tools/tg5040/BRKCHRD.pak/
SD/Tools/tg5040/.media/BRKCHRD.png
```

После резервного копирования настроек можно также удалить `SD/.userdata/tg5040/brkchrd/`.

## Диагностика

- нет картинки в карточке — проверьте регистр букв и точный путь `.media/BRKCHRD.png`;
- инструмент сразу закрывается — прочитайте `.userdata/tg5040/logs/BRKCHRD.txt`;
- настройки не сохраняются — проверьте доступность записи на SD-карту и создание `.userdata/tg5040/brkchrd/`;
- нет звука — сначала проверьте системный выход и громкость, затем параметры BRKCHRD.
