# BRKCHRD 0.5.1

## English

BRKCHRD 0.5.1 is the first fully packaged dual-platform release of the chord-performance instrument for TrimUI Brick.

### Interface fixes

- factory preset names use a dedicated larger card and no longer overflow the SOUND panel;
- percentages and other values are right-aligned with safe padding;
- the selected Tone, Body, Motion, Attack, Release or Spread value bar remains visible;
- Settings now displays `START+SELECT: SAVE AND EXIT` correctly;
- the embedded bitmap font includes the missing ampersand glyph.

### Harmony and sound

- deterministic `VOICE LEAD = OFF` and expressive `VOICE LEAD = ON`;
- exact twelve-semitone octave changes in deterministic mode;
- corrected major, minor and diminished chord extensions;
- instrument-aware Keys, Organ, Pad, Choir, Pluck, Heavy and Bass voicing profiles;
- sixteen rebuilt factory synthesis presets;
- optional PERF FX and configurable rear-button mappings.

### Packaging and identity

- official BRKCHRD logo and README banner;
- PortMaster/Knulli package with 640×480 screenshot and game cover;
- native NextUI `tg5040` Pak with `.media/BRKCHRD.png` card image;
- persistent NextUI settings stored outside the Pak;
- complete English and Russian manuals, installation guides, architecture, development, licensing, troubleshooting and hardware-test documentation.

## Русский

BRKCHRD 0.5.1 — первый полностью оформленный двухплатформенный релиз аккордового перформанс-инструмента для TrimUI Brick.

### Исправления интерфейса

- названия пресетов вынесены в отдельную крупную карточку и не вылезают из SOUND;
- проценты и другие значения выровнены справа с нормальным отступом;
- шкала выбранного Tone, Body, Motion, Attack, Release или Spread остаётся видимой;
- в Settings корректно написано `START+SELECT: SAVE AND EXIT`;
- во встроенный шрифт добавлен отсутствовавший символ амперсанда.

### Гармония и звук

- стабильный `VOICE LEAD = OFF` и живой `VOICE LEAD = ON`;
- точный октавный сдвиг на двенадцать полутонов в стабильном режиме;
- исправленные мажорные, минорные и уменьшённые расширения;
- отдельные профили раскладки Keys, Organ, Pad, Choir, Pluck, Heavy и Bass;
- шестнадцать переработанных заводских пресетов;
- отключаемый PERF FX и настраиваемый порядок задних кнопок.

### Пакеты и оформление

- официальный логотип BRKCHRD и баннер README;
- пакет PortMaster/Knulli со скриншотом 640×480 и обложкой карточки;
- нативный NextUI Pak для `tg5040` с изображением `.media/BRKCHRD.png`;
- настройки NextUI хранятся вне Pak и переживают обновление;
- полные руководства RU/EN по использованию, установке, архитектуре, разработке, лицензированию, диагностике и аппаратному тесту.
